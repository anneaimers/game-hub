# ANNE-GRACIE---GAME-HUB
PROJETO 1° BIMESTRE SUB A: SISTEMA DE JOGOS
Desenvolvido pela aluna Anne Monteiro Aimers Gracie, o GAME HUB reune 3 minijogos, sendo o primeiro de perguntas e respostas, o segundo cobra na caixa e terceiro gousmas war. Contendo menu e a possibilidade de dois players interagirem com o programa.
